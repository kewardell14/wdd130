Logos

FICTIONAL LOGO FOR AN ARCHERY STORE/RANGE (2023): This was a school assignment in which I chose a mascot and had to create a logo for a fake busisness. I chose a hawk as my mascot and created this logo for an imaginary archery store/range that specializes in custom size equipment for minors.

KUNA SCHOOL DISTRICT LOGO DESIGN (2023): I created this for the Kuna School District Logo contest. They wanted a focus on lifelong learning and a small emphasis on graduation and community.

IDAHOPE LOGO DESIGN (2023): I created this design as an entry in logo contest. Idahope had a biking project that they wanted a new logo for.

ETSY LOGO FOR WHAT IN YARNATION STORE (2022): I created this logo for my Etsy shop “What In Yarnation Store.” I sell crocheted items, hence the crochet hook and chain yarn design.

SOFTWARE STUDIO LOGO (2021): I created this logo for Kuna High School Software Studio. It features a caveman, which is the school’s mascot as well as a circuit board design to emphasis the technology aspect.

Yearbook Pages

D&D/GREEN TEAM YEARBOOK PAGE (2023)
DESCRIPTION OF PROJECT: I created the Dungeons and Dragons/Green Team (Recycling) page for Kuna High School. We had a gold and black color scheme with the theme “It’s Within Us.” I took most of these pictures myself.

S.T.R.O.N.G.E.R YEARBOOK PAGE (2022)
DESCRIPTION OF PROJECT: I created a specialty page for the Kuna High School Yearbook that showcased the S.T.R.O.N.G.E.R. qualities of being self confidence, truthful, respectful, optimistic, noble, gutsy, empathetic and responsible. I created organic shapes to stay on our theme of growth and organic elements.

Posters

ANTHONY AND CLEOPATRA Type Poster(2023):
DESCRIPTION OF PROJECT: This class room project challenged us to create a poster using all type. The waves are created from text/lines from the play.

ANTHONY AND CLEOPATRA Bradbury Thompson Poster(2023):
DESCRIPTION OF PROJECT: This classroom project was created in the style of famous graphic designer, Bradbury Thompson. He focused on using (CMYK) Cyan, Magenta, Yellow and Black, as well as overlapping duplicate objects.This class room project challenged us to create a poster using all type. The waves are created from text/lines from the play.
