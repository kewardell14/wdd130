Antony and Cleopatra Type and Bradbury Thompson Poster (2023)

Bradbury Thompson Poster: This project was created in the style of famous graphic designer, Bradbury Thompson. He focused on using (CMYK) Cyan, Magenta, Yellow and Black, as well as overlapping duplicate objects.This class room project challenged us to create a poster using all type. The waves are created from text/lines from the play.
